### 💭 ADS
> '장부리치'는 소상공인을 위한 장부 거래 디지털화 서비스로, 선결제 관리, 세금계산서 발행 및 재무 관리 기능을 통해 장부 거래의 편리성과 투명성을 높이고 사용자 간 연결성을 강화하는 서비스입니다.
